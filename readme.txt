--------------------------How To Run Program------------------------------
1. Jika belum terinstall install openpyxl dengan cara:
	open cmd, ketik pip install openpyxl
2. Ganti destinasi pada load_workbook("File Location") sesuai dengan lokasi file traintest berada
3. Ganti destinasi pada book.save("File Location") sesuai dengan lokasi file traintest berada
4. Run Program
5. Open file traintest.xlsx dan cek sheet test nilai y akan dimunculkan
